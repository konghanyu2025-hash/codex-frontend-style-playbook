# Intake Prompt

Use this prompt at the start of a frontend task.

```text
Before implementation, turn this frontend request into a concrete product and style brief.

Request:
[paste request]

Inspect the existing project first if one exists. Then produce style_brief.md with:
- product goal
- target user
- required screens or sections
- information density
- references or reference search strategy
- visual direction
- constraints
- anti goals

Do not ask about facts that can be discovered from the repo. Ask only for product choices that materially change the result.
```
