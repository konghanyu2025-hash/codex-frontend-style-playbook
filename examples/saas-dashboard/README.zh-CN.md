# 示例：SaaS 仪表盘

这个示例展示如何用本手册指导一个高信息密度的业务 SaaS 界面。

任务：

```text
为一个 B2B 收入运营工具构建仪表盘。用户是销售负责人，需要检查销售管道健康度、预测风险和下一步行动。
```

文件：

- `style_brief.zh-CN.md`：产品和风格方向。
- `visual_anchor.zh-CN.md`：生成锚点图的审查笔记。
- `implementation_brief.zh-CN.md`：具体前端规则。
- `visual_acceptance_checklist.zh-CN.md`：QA 清单。

把这个示例当作流程参考，不要当作必须复制的视觉风格。
