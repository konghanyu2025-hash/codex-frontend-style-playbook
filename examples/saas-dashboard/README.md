# Example: SaaS Dashboard

Languages: English | [简体中文](README.zh-CN.md)

This example shows how the playbook should guide a dense operational SaaS interface.

The task:

```text
Build a dashboard for a B2B revenue operations tool. Users are sales leaders who need to inspect pipeline health, forecast risk, and next actions.
```

Files:

- `style_brief.md`: product and style direction.
- `visual_anchor.md`: generated-anchor review notes.
- `implementation_brief.md`: concrete frontend rules.
- `visual_acceptance_checklist.md`: QA checklist.

Use this example as a shape reference, not as a required visual style.
