# Contributing

Language: English | [简体中文](CONTRIBUTING.zh-CN.md)

Contributions should make agent-driven frontend work more concrete, repeatable, or easier to verify.

Good contributions include:

- Better workflow steps.
- Stronger QA checks.
- Additional examples with complete briefs.
- Prompt templates that reduce vague style instructions.
- Notes for adapting the workflow to another coding agent.

## Example Requirements

New examples should include:

- `style_brief.md`
- `visual_anchor.md`
- `implementation_brief.md`
- `visual_acceptance_checklist.md`

If images are included, ensure they are original, generated for the example, or licensed for redistribution.

## Style

- Prefer concrete rules over taste words.
- Keep docs practical and directly usable by agents.
- Avoid recommending a single visual style for all products.
- Do not include proprietary screenshots, private client work, or copied brand assets.

## Release Notes

Add user-visible changes to `CHANGELOG.md`.
